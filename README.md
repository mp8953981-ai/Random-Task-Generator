**Автор:** Панов Максим

## Описание

Random Task Generator — это простое GUI-приложение на Python (Tkinter), которое помогает генерировать случайные задачи для учёбы, спорта или работы.  
Вы можете добавлять свои задачи, фильтровать по типу и просматривать историю сгенерированных заданий.  
История автоматически сохраняется в JSON-файл и загружается при запуске.

## Функции

- Генерация случайной задачи из предопределённого списка.
- Добавление новых задач с указанием типа (учеба, спорт, работа).
- Фильтрация задач по типу перед генерацией.
- Просмотр истории всех сгенерированных задач.
- Сохранение истории в `tasks.json`.
- Проверка ввода: пустая строка не допускается при добавлении задачи.

## Требования

- Python 3.11
- Стандартные библиотеки: tkinter, json, random, os
