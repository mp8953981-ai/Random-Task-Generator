**Автор:** Панов Максим 

## Описание

`Random Task Generator` — это GUI-приложение на Python, которое помогает случайным образом выбирать задачи из предопределённого или пользовательского списка.  
Поддерживает:

- Генерацию случайной задачи
- История сгенерированных задач
- Фильтрацию по типу (учеба, спорт, работа)
- Добавление новых задач с проверкой (не пустая строка)
- Сохранение / загрузка истории в JSON
- Ведение проекта в Git

## Требования

- Python 3.7+ (лучше 3.10 или 3.11)
- Стандартные библиотеки: `tkinter`, `random`, `json`, `os`

## Запуск

Склонируйте репозиторий и выполните:

```bash
python task_generator.py
```

Примечание: если в вашем проекте файл называется иначе, укажите фактическое имя.
Если вы использовали имя expense_tracker.py — замените команду на python expense_tracker.py.

Пример использования

1. Запустите приложение.
2. Нажмите «Сгенерировать задачу» — появится случайная задача.
3. Выберите фильтр: учеба, спорт или работа, чтобы видеть задачи только выбранного типа.
4. Добавьте свою задачу через поле ввода и кнопку «Добавить» (пустые строки не принимаются).
5. Закройте приложение — история автоматически сохранится в history.json. При следующем запуске она загрузится.

Возможные тесты (самопроверка)

Действие Ожидаемый результат
Нажать «Сгенерировать задачу» 3 раза В истории появятся 3 задачи
Добавить пустую строку Всплывает предупреждение, задача не добавляется
Выбрать фильтр «спорт» Отображаются только задачи с типом «спорт»
Закрыть и снова открыть приложение История остаётся прежней
